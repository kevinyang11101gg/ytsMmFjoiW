# 项目简介

## 核心功能
- 登录注册
- 蔬菜信息管理
- 水产品信息管理
- 资讯新闻
- 农产品数据可视化大屏
- 用户管理
- 系统管理

## 适用场景
- 农产品市场分析
- 农业企业决策支持
- 农产品价格趋势预测
- 农业政策制定参考

## 技术特点
- 基于 Django 框架构建，支持高效数据管理和交互
- 使用 MySQL 数据库，保证数据存储的稳定性和安全性
- 采用 Vue 前端技术，提供友好的用户界面和交互体验
- 机器学习算法应用于农产品价格预测，提高数据分析准确性
